# Задача 2

## Обозначения в python

1. ∧ (конъюнкция, логическое И) and 
2. ∨ (дизъюнкция, логическое ИЛИ) or
3. ¬ (отрицание, логическое НЕ) not
4. ≡ (тождество, эквивалентность) ==
5. → (следование, импликация) <=

## Приоритеты операций (по убыванию)

1. ==, <=
2. not
3. and
4. or

## Код

```python
print("x y w z")
for x in range(2):
    for y in range(2):
        for w in range(2):
            for z in range(2):
                f = (x and y) or (x <= z) and ((not w) == x)
                if f == 1:
                    print(x, y, w, z)
```
## Советы
1. Заключать операции в скобки, чтобы не напутать с порядком выполнения операция (а сравнение вида x == not y вообще выдаст ошибку)
2. После изменения порядка вывода переменных в print() не забыть поменять порядок в первой строке программы
